## Project Description
The **Birthday Event Registration System** is a web-based application designed to simplify the process of registering guests for birthday celebrations. It provides organizers with a centralized platform to manage invitations, track RSVPs, and maintain guest lists efficiently.  

**Purpose:**  
- To automate birthday event registration and reduce manual tracking.  
- To provide quick access to guest information for organizers.  

**Problem Solved:**  
Traditional guest registration methods often involve paper lists or scattered spreadsheets, which can be error-prone and time-consuming. This system addresses those issues by offering a secure, user-friendly, and integrated solution.

## Functionalities
- **Add Data** – Register new guests for the birthday event.  
- **View Data** – Display guest lists and RSVP details.  
- **Update Data** – Edit guest information or event details.  
- **Delete Data** – Remove outdated or incorrect entries.  
- **API Integration** – Connect with external services (e.g., email notifications, SMS reminders).  

## Technologies Used
- **Frontend:** HTML, Bootstrap  
- **Client-Side Scripting:** JavaScript (Fetch API)  
- **Backend:** PHP (PDO for secure database interaction)  
- **Database:** MySQL  
- **Version Control:** GitHub
